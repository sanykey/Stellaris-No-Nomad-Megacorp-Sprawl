# Project context

- The installed Stellaris game files are located at `D:\apps\Steam\steamapps\common\Stellaris`.
- The user's own and manually installed Stellaris mods are located at `C:\Users\snky\Documents\Paradox Interactive\Stellaris\mod`.
- The same mods directory contains Steam Workshop mod descriptors whose filenames begin with `ugc_`. These descriptor files can be used to find the corresponding Workshop mod directories.
- See `README.md` for the description and intended behavior of this mod.
- The mod namespace is `nnms`; prefix mod files with `nnms_`.

Treat the game installation and other mods as reference sources unless the user explicitly asks to modify them. Make changes for this project inside the current mod directory.
